# Create React Liferay - JetBrains Plugin

![Plugin Icon](https://raw.githubusercontent.com/Benhur-Silveira/create-react-liferay-plugin/main/images/icone.png)

A professional scaffolding tool for Liferay DXP React modules, built specifically for JetBrains IDEs.

## 🚀 Overview
**Create React Liferay** streamlines the development of React-based modules and widgets within the Liferay DXP ecosystem. It brings the power of a modern generator directly into your favorite IDE, ensuring consistent project structures and high productivity.

## ✨ Features
- **One-Click Scaffolding**: Generate full React modules without leaving the IDE.
- **Tools Menu Integration**: Quick access via `Tools` -> `React Liferay: Criar Módulo`.
- **Customizable Settings**: Fine-tune the generator's behavior in the IDE settings.
- **Liferay Best Practices**: Automatically generates files following standard module structures.

## 🛠 Usage
1. Open your Liferay Workspace.
2. Navigate to **Tools** > **React Liferay: Criar Módulo**.
3. Fill in the required module information (name, description, etc.).
4. Your new module will be created and ready for development.

## ⚙ Configuration
Configure the plugin behavior under:
**Settings/Preferences** > **Tools** > **Create React Liferay**

## 📦 Requirements
- IntelliJ IDEA (Community or Ultimate) / WebStorm / Android Studio.
- Version 2024.1 or higher recommended.
- Java 21+ for building.
